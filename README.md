# Pychatting 

Chat application built in python

# Technologies 

- FastAPI 
- Docker
- SqlLite
- uv

# Docker Commands

## Make Commands
``
make c-up		# This will start the container and build the image
``
</br>
``
make c-down		# Clean up (Keeps image)
``
</br>
``
make c-down-rmi # Clean up and removes images
``


## Standard Docker Commands

``
docker compose up     # Creates the image and exposes to the 8080 port
``

</br>

``
docker compose down     # Clean up
``
